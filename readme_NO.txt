Dette skriptet vil opprette alle mapper og sette variabler av selg selv.
Ønsker du å endre hvor GOPATH blir satt kan du endre linje to (C:\Go\Work)
til det du måtte ønske.

Hvis du har OS X eller Linux, bruk InstallGo.sh
Kjør scriptet i terminal med sh InstallGo.sh
Eller legg det i en mappe og start det som et vanlig program
På Windows så bruker du InstallGo.bat
